# Student-Management-System

##  Description
A simple Java console-based application to manage student records.

##  Features
- Add student
- View all students
- Search student by ID
- Delete student

##  Technologies Used
- Java
- OOP Concepts
- ArrayList

##  How to Run
1. Compile:
   javac Student.java StudentManagement.java

2. Run:
   java StudentManagement

##  Output
Console-based menu driven program
